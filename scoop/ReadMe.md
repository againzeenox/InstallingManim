# Install `Scoop`
#### 1. Run in powershell
````PS
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
````
#### 2. Run in powershell
````PS
irm get.scoop.sh | iex
````

# Install `python` + `ffmpeg`

#### Run:
````PS
scoop install python ffmpeg
````

# Install `ManimCE`
#### Run:
````PS
python -m pip install manim
````
